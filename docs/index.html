Install & smoke test
Set up the core library, CLI, and plugin in a few commands.
  • 1 Install the core library and CLI on the machine running OpenClaw.
  • 2 Clone the plugin repo and add it to your OpenClaw config.
  • 3 Restart OpenClaw and verify memories are captured & recalled. 
# Core + CLI
npm install @akashabot/openclaw-memory-offline-core
npm install -g @akashabot/openclaw-mem

# Plugin (example path)
cd C:\Users\<you>\clawd\repos
 git clone https://github.com/AkashaBot/openclaw-memory-offline-sqlite-plugin.git

# Smoke test
openclaw-mem --db memory.sqlite init
openclaw-mem remember "First memory" --db memory.sqlite
openclaw-mem search "First" --db memory.sqlite --hybrid
Configure in OpenClaw
Drop‑in memory plugin for your agents.
{
  "plugins": {
    "load": {
      "paths": [
        "C:\\Users\\<you>\\clawd\\repos\\openclaw-memory-offline-sqlite-plugin"
      ]
    },
    "slots": {
      "memory": "memory-offline-sqlite"
    },
    "entries": {
      "memory-offline-sqlite": {
        "enabled": true,
        "config": {
          "dbPath": "C:\\Users\\<you>\\.openclaw\\memory\\offline.sqlite",
          "autoRecall": true,
          "autoCapture": true,
          "mode": "hybrid",
          "topK": 5,
          "candidates": 50,
          "semanticWeight": 0.7,

          // Provider (ollama | openai)
          "provider": "ollama",
          "ollamaBaseUrl": "http://127.0.0.1:11434",
          "embeddingModel": "bge-m3",
          "ollamaTimeoutMs": 3000
        }
      }
    }
  }
}
